# 📚 GUIDE D'UTILISATION COMPLET - HOSPITSIS

## 🎯 BIENVENUE DANS HOSPITSIS

**HospitSIS** est une plateforme complète de gestion hospitalière conçue pour simplifier et optimiser tous les processus médicaux et administratifs. Que vous soyez patient, médecin, infirmier, administrateur ou super administrateur, ce guide vous accompagnera pas à pas dans l'utilisation de l'application.

---

## 🚪 COMMENT ACCÉDER À L'APPLICATION

### Étape 1 : Ouvrir votre navigateur web
- Lancez votre navigateur préféré (Chrome, Firefox, Safari, Edge)
- Tapez l'adresse de l'application dans la barre d'adresse

### Étape 2 : Page d'accueil
- Vous arrivez sur la page d'accueil publique
- Cliquez sur "Se connecter" ou "S'inscrire" selon votre besoin

### Étape 3 : Sélection du portail
- Choisissez votre type d'utilisateur :
  - **Patient** : Pour les patients
  - **Personnel** : Pour le staff hospitalier
  - **Médecin Externe** : Pour les médecins indépendants

---

## 👤 GUIDE D'UTILISATION PAR TYPE D'UTILISATEUR

### 1. 📋 POUR LES PATIENTS

#### **Première utilisation : Création de compte**
1. Allez sur la page d'accueil
2. Cliquez sur "S'inscrire" → "Portail Patient"
3. Remplissez le formulaire :
   - Nom complet
   - Email
   - Numéro de téléphone
   - Mot de passe
4. Cliquez sur "Créer mon compte"
5. Vérifiez votre email pour activer le compte

#### **Connexion à votre compte**
1. Allez sur `/portal/login`
2. Entrez votre email et mot de passe
3. Cliquez sur "Se connecter"

#### **Prendre un rendez-vous**
1. Connectez-vous à votre compte patient
2. Cliquez sur "Prendre rendez-vous" ou "Nouveau RDV"
3. Sélectionnez l'hôpital souhaité
4. Choisissez le service médical
5. Sélectionnez le médecin disponible
6. Choisissez une date et heure disponible
7. Confirmez le rendez-vous
8. Recevez une confirmation par email

#### **Consulter vos informations médicales**
- **Historique médical** : Tous vos anciens rendez-vous et diagnostics
- **Prescriptions** : Vos ordonnances actuelles et passées
- **Documents** : Téléchargez vos résultats d'examens
- **Factures** : Consultez et payez vos factures en ligne

#### **Communiquer avec l'hôpital**
- Utilisez la messagerie intégrée pour contacter votre médecin
- Recevez des rappels de rendez-vous
- Consultez vos contacts d'urgence

---

### 2. 👨‍⚕️ POUR LES MÉDECINS INTERNES

#### **Connexion**
1. Allez sur `/select-hospital`
2. Choisissez votre hôpital
3. Entrez vos identifiants (fournis par l'administrateur)
4. Vous êtes automatiquement redirigé vers votre tableau de bord

#### **Votre tableau de bord quotidien**
- **Patients du jour** : Liste de vos rendez-vous
- **Observations critiques** : Alertes sur l'état des patients
- **Prescriptions en attente** : À signer ou valider
- **Statistiques** : Vos performances médicales

#### **Gérer les consultations**
1. Cliquez sur un patient dans votre liste
2. Consultez son dossier médical
3. Ajoutez vos observations cliniques
4. Prescrivez des médicaments si nécessaire
5. Créez une ordonnance électronique
6. Signez numériquement la prescription

#### **Admettre un patient**
1. Lors d'une consultation, si nécessaire :
2. Cliquez sur "Admettre le patient"
3. Choisissez le type d'admission (urgence/standard)
4. Sélectionnez un lit disponible
5. Remplissez les informations médicales
6. Validez l'admission

---

### 3. 👩‍⚕️ POUR LES INFIRMIERS(ÈRES)

#### **Connexion et tableau de bord**
1. Connectez-vous via le portail staff
2. Votre dashboard affiche :
   - Tâches quotidiennes assignées
   - Constantes vitales à prendre
   - Médicaments à administrer
   - Alertes patient

#### **Prendre les constantes vitales**
1. Cliquez sur "Nouvelles observations"
2. Sélectionnez le patient
3. Choisissez le type d'observation :
   - Température
   - Tension artérielle
   - Fréquence cardiaque
   - Saturation en oxygène
4. Entrez les valeurs mesurées
5. Ajoutez des notes si nécessaire
6. Sauvegardez l'observation

#### **Administrer les médicaments**
1. Consultez la liste des prescriptions
2. Vérifiez l'identité du patient
3. Confirmez l'administration du médicament
4. Enregistrez l'action avec l'heure

---

### 4. 👨‍💼 POUR LE PERSONNEL ADMINISTRATIF

#### **Gestion des patients**
1. Allez dans "Gestion des patients"
2. Recherchez un patient par nom ou numéro
3. Créez un nouveau dossier patient
4. Modifiez les informations personnelles
5. Archivez les dossiers anciens

#### **Organisation des rendez-vous**
1. Consultez le planning général
2. Ajoutez/modifiez/annulez des RDV
3. Gérez les conflits d'horaire
4. Envoyez des rappels aux patients

#### **Gestion des factures**
1. Générez des factures pour les prestations
2. Consultez les paiements en attente
3. Marquez les factures comme payées
4. Exportez les rapports financiers

---

### 5. 💰 POUR LES CAISSIERS

#### **Validation des paiements**
1. Connectez-vous à votre dashboard caissier
2. Consultez les "Paiements en attente"
3. Pour chaque rendez-vous :
   - Vérifiez l'identité du patient
   - Confirmez le paiement
   - Imprimez le reçu
   - Marquez comme payé

#### **Consultations sans rendez-vous**
1. Cliquez sur "Consultation sans RDV"
2. Enregistrez le patient (s'il n'existe pas)
3. Sélectionnez le service demandé
4. Calculez le tarif
5. Validez le paiement
6. Imprimez le ticket de consultation

#### **Gestion des reçus**
- Imprimez les reçus de paiement
- Consultez l'historique des transactions
- Gérez les remboursements si nécessaire

---

### 6. 🔵 POUR LES MÉDECINS EXTERNES

#### **Première inscription**
1. Allez sur `/medecin/externe/login`
2. Cliquez sur "S'inscrire" ou allez sur `/medecin/externe/register`
3. Remplissez le formulaire d'inscription :
   - Informations personnelles (nom, prénom, spécialité)
   - Coordonnées et qualifications
   - Documents justificatifs
4. Soumettez votre demande d'inscription
5. **Attendez la validation** : Votre compte doit être approuvé par un Super Admin

#### **Connexion à votre compte**
1. Allez sur `/medecin/externe/login`
2. Entrez votre email et mot de passe
3. Cliquez sur "Se connecter"
4. Vous accédez à votre tableau de bord externe

#### **Votre tableau de bord**
- **Statut de disponibilité** : En ligne/hors ligne
- **Solde du portefeuille** : Montant disponible
- **Rendez-vous du jour** : Consultations programmées
- **Commissions du mois** : Revenus générés
- **Messages patients** : Communications en attente

#### **Gérer votre disponibilité**
1. Sur votre tableau de bord, cliquez sur le bouton "Disponible"
2. Basculez entre "Disponible" et "Indisponible"
3. Quand vous êtes disponible, les patients peuvent vous contacter
4. Quand indisponible, vous ne recevez plus de nouvelles demandes

#### **Gérer vos patients**
1. Allez dans "Mes Patients"
2. Consultez la liste de vos patients assignés
3. Cliquez sur un patient pour voir son profil
4. Accédez aux dossiers médicaux partagés

#### **Créer et gérer des prestations**
1. Allez dans "Prestations"
2. **Créer une nouvelle prestation :**
   - Choisissez la catégorie (consultation, visite, téléconsultation)
   - Définissez le prix
   - Ajoutez une description
   - Activez/désactivez la prestation
3. **Modifier une prestation existante**
4. **Supprimer une prestation** (si elle n'est pas utilisée)

#### **Gérer vos rendez-vous**
1. Allez dans "Rendez-vous"
2. Consultez vos consultations programmées
3. Acceptez ou refusez les demandes de rendez-vous
4. Modifiez les horaires si nécessaire
5. Annulez des rendez-vous en cas d'imprévu

#### **Créer des prescriptions**
1. Lors d'une consultation, allez dans "Prescriptions"
2. Cliquez sur "Nouvelle Prescription"
3. Sélectionnez le patient
4. Ajoutez les médicaments :
   - Nom du médicament
   - Dosage et fréquence
   - Durée du traitement
   - Instructions spéciales
5. Signez numériquement la prescription
6. Envoyez-la au patient

#### **Gérer votre portefeuille**
1. Allez dans "Recharger"
2. Choisissez le montant à recharger
3. Sélectionnez le mode de paiement (CinetPay)
4. Validez le paiement
5. Votre solde est mis à jour automatiquement

#### **Suivre vos commissions**
- **Tableau de bord** : Commissions du mois en cours
- **Historique** : Toutes vos transactions passées
- **Calcul automatique** : Selon les tranches de prix configurées
- **Paiements** : Réception automatique sur votre portefeuille

#### **Gérer votre profil**
1. Allez dans "Profil"
2. Mettez à jour vos informations personnelles
3. Modifiez vos qualifications et spécialités
4. Changez votre mot de passe
5. Téléchargez vos documents justificatifs

---

### 7. 👑 POUR LES ADMINISTRATEURS D'HÔPITAL

#### **Gestion des utilisateurs**
1. Allez dans "Gestion des utilisateurs"
2. Créez des comptes pour le personnel :
   - Médecins, infirmiers, administratifs, caissiers
   - Définissez leurs rôles et permissions
   - Assignez-les à des services
3. Activez/désactivez des comptes
4. Réinitialisez les mots de passe

#### **Configuration de l'hôpital**
1. Paramètres généraux : nom, adresse, logo
2. Gestion des services médicaux
3. Configuration des tarifs
4. Paramètres de sécurité (2FA)

#### **Supervision et rapports**
- Consultez les logs d'audit
- Générez des rapports d'activité
- Surveillez les performances
- Gérez les abonnements

---

### 8. 🏢 POUR LES SUPER ADMINISTRATEURS

#### **Accès spécial au Super Admin**
**URL d'accès :** `/superadmin/login`

**⚠️ INFORMATION IMPORTANTE :**
Le Super Admin dispose d'un système de sécurité renforcé avec **code d'accès secret**. Voici comment accéder :

##### **Étape 1 : Connexion initiale**
1. Allez sur `/superadmin/login`
2. Entrez votre email et mot de passe
3. Cliquez sur "Se connecter"

##### **Étape 2 : Vérification du code secret**
Après la connexion, vous devez entrer un **code d'accès secret de 8 caractères**.
- Ce code vous a été fourni lors de votre configuration initiale
- Il est unique à votre compte Super Admin
- Conservez-le en sécurité !

##### **Étape 3 : Accès au panneau Super Admin**
Une fois vérifié, vous accédez au `/admin-system/dashboard`

#### **Fonctionnalités du Super Admin**

##### **🏥 Gestion Globale des Hôpitaux**
1. **Créer un nouvel hôpital :**
   - Cliquez sur "Créer Hôpital"
   - Remplissez : nom, adresse, admin principal
   - L'admin reçoit automatiquement ses identifiants

2. **Superviser tous les hôpitaux :**
   - Liste de tous les hôpitaux actifs/inactifs
   - Statistiques par hôpital
   - Activation/désactivation d'hôpitaux

##### **👨‍⚕️ Gestion des Médecins Externes**
1. **Valider les inscriptions :**
   - Consultez les demandes en attente
   - Vérifiez les qualifications
   - Approuvez ou rejetez l'inscription

2. **Gestion des portefeuilles :**
   - Bloquer/débloquer des comptes
   - Ajuster manuellement les soldes
   - Prélever des commissions

##### **💰 Gestion Financière Globale**
1. **Plans d'abonnement :**
   - Créer/modifier les forfaits hospitaliers
   - Définir les prix et fonctionnalités
   - Gérer les renouvellements

2. **Règles de commission :**
   - Configurer les taux par tranches de prix
   - Ajuster les frais d'activation
   - Calcul automatique des commissions

3. **Monitoring financier :**
   - Revenus totaux du système
   - Commissions perçues
   - Transactions récentes
   - Factures impayées

##### **📊 Tableaux de Bord Super Admin**
- **Métriques globales :** Nombre d'hôpitaux, utilisateurs, patients
- **Revenus :** Abonnements + Commissions + Frais d'activation
- **Statistiques mensuelles :** Évolution des revenus
- **Alertes :** Hôpitaux en attente, spécialistes à valider

##### **🔧 Actions Administratives**
- **Test des fonctionnalités :** Boutons de test pour les paiements
- **Logs système :** Suivi de toutes les actions
- **Configuration globale :** Paramètres système
- **Support utilisateurs :** Accès aux demandes de support

## 🔐 SÉCURITÉ ET BONNES PRATIQUES

### **Pour tous les utilisateurs :**
- **Ne partagez jamais vos identifiants**
- **Déconnectez-vous après utilisation**
- **Utilisez des mots de passe forts**
- **Activez la 2FA si disponible**

### **Pour les professionnels de santé :**
- **Vérifiez toujours l'identité des patients**
- **Signez numériquement vos prescriptions**
- **Respectez la confidentialité des données médicales**

### **Pour les Super Admins :**
- **Conservez le code d'accès secret en sécurité**
- **Vérifiez les actions avant validation**
- **Surveillez régulièrement les logs d'audit**

## 🆘 DÉPANNAGE COURANT

### **Problème : Mot de passe oublié**
1. Sur la page de connexion, cliquez "Mot de passe oublié"
2. Entrez votre email
3. Suivez les instructions reçues par email

### **Problème : Accès refusé**
- Vérifiez que votre compte est actif
- Contactez votre administrateur
- Pour Super Admin : vérifiez le code d'accès

### **Problème : Application lente**
- Rafraîchissez la page
- Videz le cache de votre navigateur
- Contactez le support technique
